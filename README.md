# GD VST

A mod to add VST plugin support to Geometry Dash.

## Building

You have to download the JUCE library yourself and put it in the root project folder.

Additionally, you have to manually download and provide the VST2 sdk (you can simply paste the vst2.x folder inside JUCE/modules/juce_audio_processor_headless/format_types/VST3_SDL/plugininterfaces).
