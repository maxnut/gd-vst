# GD VST
This mod allows you to modify the in-game audio with audio plugins!

## Usage

Press the toggle keybind to open the plugin manager window (default keybind is V),
then right click the window to add a new plugin.

The mod will scan the plugin directory for any audio plugin.
You can access the directory from the plugin selector by going into File > Open Plugin Folder

You can disable/enable plugins with the checkbox, or reorder/remove them by right clicking.

## Disclaimer

<cr>The mod does not come with **any** plugin by default! You have to get them yourself.</cr>
